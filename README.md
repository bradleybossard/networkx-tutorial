### Setup from scratch
```

conda activate networkx-tutorial
conda install --yes networkx
conda env export --file environment.yaml
```

### Setup using environment.yaml file
```
conda create --name networkx-tutorial --file environment.yaml
conda activate networkx-tutorial
```
